# PCAT
- How to init npm :
- `npm init -y` | -y => Default options
- How to install prettier:  
`npm install prettier -D --save-exact`<br>
.prettierc file : 
~~~json
{
  "tabWidth": 2,
  "useTabs": false,
  "semi":true,
  "singleQuote": true,
  "trailingComma": "es5"
}
~~~
